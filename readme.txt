Программа обучает классификатор (бинарный или многоклассовый) методом опорных векторов.
В качестве признаков рассматривается гистограмма ориентированных градиентов и цветовые признаки.
Используется пирамида десктипторов и нелинейное ядро SVM (n = 1, L = 0,32).

Обучить модель:
./task2 -d ../../data/binary/train_labels.txt -m model.txt --train   #бинарный классификатор
./task2 -d ../../data/multiclass/train_labels.txt -m model_multiclass.txt --train   #многоклассовый классификатор

Классифицировать изображения из тестовой выборки с помощью обученного классификатора:
./task2 -d ../../data/binary/test_labels.txt -m model.txt -l predictions.txt --predict
#многоклассовый - аналогично

Проверка точности классификации:
./compare.py data/binary/test_labels.txt build/bin/predictions.txt
#многоклассовый - аналогично
